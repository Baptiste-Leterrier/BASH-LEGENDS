# BASH-LEGENDS 🎮

A gacha-style game to learn about log parsing, `jq`, `awk`, and other command-line text processing tools with progression and pulling mechanics.

**Play now:** [https://baptiste-leterrier.github.io/BASH-LEGENDS/](https://baptiste-leterrier.github.io/BASH-LEGENDS/)

## 📖 About

BASH-LEGENDS transforms learning command-line tools into an engaging, gamified experience. Instead of dry tutorials, you'll:
- **Pull** legendary BASH tools and skills through a gacha-style system
- **Parse** logs and data to complete challenges
- **Master** tools like `jq`, `awk`, `grep`, and more
- **Progress** through difficulty levels and unlock new abilities

Whether you're a beginner wanting to learn text processing or an experienced user looking to sharpen your skills, BASH-LEGENDS makes it fun and rewarding.

## 🎯 Features

- **Gacha Pulling System**: Collect legendary BASH tools and skills with varying rarities
- **Progressive Learning**: Start with basics and advance to complex log parsing challenges
- **Interactive Challenges**: Real-world log parsing scenarios to practice on
- **Skill Development**: Learn `jq`, `awk`, `grep`, `sed`, and other essential tools
- **Rewards & Progression**: Earn currency, unlock achievements, and climb the ranks

## 🚀 Getting Started

Simply visit the app at [https://baptiste-leterrier.github.io/BASH-LEGENDS/](https://baptiste-leterrier.github.io/BASH-LEGENDS/) to start playing!

### How to Play

1. **Start Your Journey**: Create your profile and begin with the tutorial
2. **Pull for Skills**: Use your currency to summon new BASH tools and techniques
3. **Complete Challenges**: Use your acquired skills to parse logs and solve puzzles
4. **Earn Rewards**: Collect currency, experience, and rare tools
5. **Master the Arts**: Progress through levels and become a BASH legend

## 🛠️ Tools Covered

- **jq**: JSON querying and manipulation
- **awk**: Text pattern scanning and processing
- **grep**: Pattern matching in files
- **sed**: Stream editing and text transformation
- **cut**: Extract columns from text
- **sort & uniq**: Organize and deduplicate data
- And more!

## 💡 Why BASH-LEGENDS?

Traditional learning can be boring. BASH-LEGENDS gamifies the learning experience by:
- Making each new tool feel like a valuable pull
- Creating engaging, real-world scenarios
- Providing immediate feedback and progression
- Building a sense of achievement as you master tools
